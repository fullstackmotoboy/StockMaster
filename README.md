# StockMaster - Atacadão dos Hardwares 

O **StockMaster** é um sistema de gerenciamento de estoque desenvolvido para o projeto "Atacadão dos Hardwares". O objetivo é controlar a entrada e saída de peças de computadores, periféricos e componentes técnicos.

## Tecnologias
- **Linguagem:** Python 3
- **Ambiente:** Venv (Ambiente Virtual Isolado)
- **Interface:** HTML5, CSS3 e JavaScript
- **Controle de Versão:** Git & GitHub
